**Link:** https://github.com/nuxt/ui/issues/2968

## Setup
```sh
# Init
npx nuxi@latest init -t ui3 template

# Setup
pnpm install

# Development Server
pnpm run dev
# Start the development server on `http://localhost:3000`:

# Build
pnpm run build

# Preview
pnpm run preview
```
